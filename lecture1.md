### 算法
1. 输入、输出、有穷、确定、有效！
2. 算法分析：
   1. 时间复杂度 -- 最关注最高次数！通常指最坏时间复杂度
    ```Python
        i = 1
        j = 0
        while (i + j <= n):
            if(i > j):
                j++
            else:
                i++;
    ```
    + 此代码的if语句执行n次, 算法时间复杂度 O(n)
    ```C
        int time(int n)
        {
            int cnt = 0, x = 2;
            while(x < n/2)
            {
                cnt++;
                x *= 2;
            }
            return cnt;
        }
    ```
    + 此代码的算法时间复杂度 O(log n)(省略底数2)
